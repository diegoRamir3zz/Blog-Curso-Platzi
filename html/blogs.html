Tecnologias principales

En este blog hablare de las tecnologias pricipales front-end y lo que yo he aprendido sobre ellas.

Blogs Principales

HTML: Maquetar

HTML es el lenguaje más basico y con el que por defecto construimos paginas web, con el podemos maquetar la estructura general de nuestra pagina web, entre muchas otra cosas...

CSS: Estilos

Con CSS podras darle estilos al maquetado que crees con html. En todas las paginas que ingreses veras el uso de css, pues nos permite personalizar la web a nuestro acomodo.

JAVASCRIPT: Interacción

JAVASCRIPT es la herramienta que nos ayuda a darle la interacción a nuestro sitio web, y nos permite acer paginas mucho más dinamicas y entretenidas para los usuarios.
